# @crash/shared

Pacote compartilhado do monorepo: contratos de eventos RabbitMQ, envelope tipado, routing keys, serializacao de centavos e contratos WebSocket (Socket.IO).

## Modulos

- **Eventos RabbitMQ** — tipos, payloads, routing keys e envelope (`EventEnvelope`)
- **WebSocket** — `WsEventTypes`, payloads de rodada (`round-events.ts`) e apostas (`bet-events.ts`)

Export central: `packages/shared/src/index.ts`.
